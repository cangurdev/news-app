# News App

Multicamp Flutter kampı bitirme projesi.

Özellikler					

- Splash sayfası					
- Giriş ve Kayıt ol sayfaları
- Ana ekranda haberler listesi 						
- Haber detay sayfası (Webview ile açılıyor)
- Ayarlar ekranı (Uygulama versiyonu, Çıkış Yap, Geliştirici Hakkında)					
- Uygulama İkonu					
- Lisanslar sayfası
